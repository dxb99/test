DXB99 APP GUIDE

HOW TO USE THE APP

Pick a topic below, or search for a word like maps, history, save, admin, or matchup.

Select a guide topic

MATCHUP TAB

This is the main screen. It shows the current matchup, the two teams, team skill totals, the skill difference, and the session maps.

  1. If there is no matchup, press CLICK TO GET STARTED. This takes you to the Generator tab.
  2. If there is a matchup, read the red team and blue team from this screen.
  3. The timer shows when the current matchup expires.
  4. The Session Maps card shows the maps planned for the session.
  5. The highlighted Match Maker name shows who is currently responsible for making teams.

Simple rule: this tab is for viewing what everyone should play right now.

GENERATOR TAB

This is where you create balanced team options.

  1. Pick the Match Maker from the dropdown.
  2. Make sure the players who are playing are checked.
  3. Click CLICK TO GENERATE MATCHUP.
  4. The app shows possible teams, sorted by the smallest skill difference first.
  5. Click CLICK TO SELECT on the matchup you want to use.
  6. After saving, the app takes you back to the Matchup tab.

If teams are uneven, the Blitz option can show matchups where the smaller team has more skill points.

Simple rule: generate options, then always select the one you want.

HISTORY TAB

This tab shows old matchups that were selected.

  1. By default, the app shows recent history so it loads faster.
  2. Click a history row to open the full match card.
  3. The match card shows both teams, skill totals, difference, and maps played for that session.
  4. Use LOAD ALL HISTORY if you need older matches.
  5. Use CLEAR MATCHUP HISTORY only if you really want to erase history and reset the current matchup.

If session maps need correction, use the edit map controls inside the opened history card, then save the changes.

Simple rule: history is where you check what happened before.

PLAYERS TAB

This tab is for managing players. It is protected by admin mode.

  1. Unlock admin mode first.
  2. Click ADD PLAYER to add a new player.
  3. Edit a player name directly in the table.
  4. Use the skill controls to adjust a player's skill number.
  5. Use the Match Maker checkbox to choose who can appear in the Match Maker list.
  6. Click SAVE when done.

Simple rule: if you change players, click SAVE before leaving.

MAP LIST TAB

This tab controls the maps for the session.

  1. Click GENERATE SESSION MAPS to create the next map list.
  2. The left side shows the maps for this session.
  3. The right side shows the complete map list.
  4. If a map was skipped, click the X next to that map.
  5. After the session is done, click SAVE SESSION PROGRESS.
  6. The app remembers what was played and brings skipped maps back later.

Bonus is connected to Elimination. Bonus maps count as extra Elimination maps.

Use BUILD CUSTOM SESSION if you want to pick maps yourself. Use COPY SESSION MAPS to copy the session map card as an image.

Simple rule: before saving progress, only maps that were played should still be showing.

HELP TAB

  1. Click a tab name at the top of this guide to jump to that section.
  2. Use the search box to find words like player, maps, save, history, or admin.
  3. If nothing shows up, clear the search box and try a simpler word.

Simple rule: come here when you forget what a button does.
